# 🤖 Shramic AI Assistant

An AI-powered multilingual chatbot developed for **Shramic Networks** to provide instant information about the company, internships, services, leadership, contact details, and workforce solutions.

---

**Backend API (Render):**

https://shramic-ai-assistant.onrender.com

## 🏢 About Shramic Networks

Shramic Networks is an AI-powered infrastructure platform focused on empowering India's rural workforce by connecting workers, farmers, contractors, businesses, and communities through technology-driven solutions.

### Mission

To create employment opportunities, improve rural livelihoods, provide fair wages, and connect workers, farmers, experts, and businesses through digital infrastructure.

### Vision

To build a national rural employment and services network powered by technology, data intelligence, and transparent workforce systems.

## ⚙️ Local Installation

### Clone Repository

```bash
git clone https://github.com/ArjunaNagarjuna/shramic-ai-assistant.git
```

### Backend Setup

```bash
pip install -r requirements.txt
uvicorn backend.api:app --reload
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
